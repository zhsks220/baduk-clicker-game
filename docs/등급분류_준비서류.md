# 게임물관리위원회 등급분류 신청 준비서류

## 📋 필요 서류 체크리스트

### 1. ✅ 게임물내용설명서 (필수) - **준비 가능**
- **형식**: 한글(.hwp) 또는 MS-Word(.docx)
- **포함 내용**:
  - [ ] 게임 줄거리/시나리오 요약
  - [ ] 주요 캐릭터 설명
  - [ ] 아이템 리스트 및 시스템 설명
  - [ ] 게임 조작방법
  - [ ] 주요 게임 진행 장면 스크린샷
  - [ ] 점수/골드 획득 방법
  - [ ] 게임 종료 방법

### 2. ✅ 실행 가능한 게임물 (필수) - **준비 가능**
- `npm run build`로 생성된 dist 폴더
- 또는 접속 가능한 URL 제공

### 3. ✅ 게임 플레이 동영상 (필수) - **준비 가능**
- **분량**: 30분 내외
- **형식**: mpg, avi, asf 등
- **포함 내용**:
  - [ ] 연령등급 표시 화면
  - [ ] 스토리 인트로
  - [ ] 메인 게임 플레이 (클릭, 바둑돌 파괴)
  - [ ] 업그레이드 시스템
  - [ ] 상점 시스템
  - [ ] 미션 시스템
  - [ ] 자동 클릭 시스템
  - [ ] 더보기 메뉴
  - [ ] 종료 확인 모달

### 4. ⏳ 사업자 서류 (필수) - **별도 준비 필요**
다음 중 하나:
- [ ] 게임물제작업자등록증
- [ ] 게임물배급업자등록증
- [ ] 개인사업자등록증

### 5. ⏳ 심의 수수료 입금증 (필수) - **별도 준비 필요**
- **입금계좌**: 국민은행 011201-04-078963
- **예금주**: 게임물관리위원회
- [ ] 입금 후 무통장 입금증 캡처

### 6. ⏳ 초상권/라이센스 서류 (해당시) - **해당 없음**
- 본 게임은 자체 제작 캐릭터 사용으로 불필요

## 📁 지금 바로 준비 가능한 것

1. **게임물내용설명서** - Claude가 작성 가능
2. **게임 빌드 파일** - `npm run build` 실행
3. **스크린샷** - Puppeteer로 촬영 가능
4. **플레이 동영상** - 화면 녹화 필요 (OBS, 윈도우 게임바 등)

---

## 📝 신청 절차

1. [게임물관리위원회](https://www.grac.or.kr) 접속
2. [등급분류신청] → [오픈마켓/등급분류 신청] 선택
3. 플랫폼: **모바일 게임** 선택
4. 스토어명: **기타-앱인토스** 입력
5. 서류 업로드 및 수수료 입금
6. 심의 대기 (약 10~15일)

---

## 💰 예상 비용

- 심의 수수료: 약 10만원 내외 (게임 종류에 따라 상이)
- 정확한 금액은 GRAC 사이트에서 확인

# 게임물관리위원회 등급분류 신청 가이드

## 개요

모바일 게임(앱인토스)은 **"오픈마켓 등급분류 간소화"** 방식으로 신청합니다.

---

## 신청 절차

### 1단계: 회원가입
- **사이트**: https://www.grac.or.kr
- **회원 유형**: 개인회원 (사업자 없어도 가능)

### 2단계: 오픈마켓 등급분류 신청
1. 로그인
2. "오픈마켓 등급분류 신청" 클릭
3. 스토어명: **"기타"** 선택 → **"앱인토스"** 입력

### 3단계: 설문 응답 (간소화 방식)
- 폭력성, 선정성, 사행성 등 설문에 응답
- 시스템이 자동으로 등급 산정
- 예상 등급: **전체이용가**

### 4단계: 서류 업로드

| 서류 | 설명 | 준비 상태 |
|------|------|----------|
| 게임물 내용 설명서 | 시나리오, 캐릭터, 시스템 설명 문서 | 📝 작성 필요 |
| 실행 가능한 게임물 | 빌드 파일 또는 접속 가능한 URL | ✅ `npm run build` |
| 플레이 동영상 | 30분 내외, 모든 주요 기능 포함 | 🎬 촬영 필요 |

### 5단계: 수수료 납부
- **납부 방법**: 무통장입금 또는 카드결제
- **입금계좌**: 국민은행 011201-04-078963 (예금주: 게임물관리위원회)

### 6단계: 심의 대기
- **소요 기간**: 약 3~7일 (간소화 방식)
- **결과 확인**: 이메일 및 홈페이지에서 확인
- **발급물**: 등급분류번호

---

## 수수료 안내

| 구분 | 금액 | 비고 |
|------|------|------|
| 기본 수수료 | 약 5~10만원 | 게임 종류에 따라 상이 |
| 50인 미만 기업 | **50% 감면** | 환급 신청 가능 |
| 오픈마켓 운영사 대행 | **30% 감면** | - |
| 비영리 게임물 | **면제** | 수익 없는 경우 |

## 우리 게임 정보

| 항목 | 내용 |
|------|------|
| 게임명 | 바둑돌 부수기 |
| 제작사 | (개인/스튜디오명 입력) |
| 플랫폼 | 모바일 웹게임 |
| 스토어 | 앱인토스 (기타) |
| 예상 등급 | 전체이용가 |
| 내용정보 | 폭력성 없음, 선정성 없음, 사행성 없음 |

## 필요 서류 상세

### 1. 게임물 내용 설명서
- **형식**: 한글(.hwp) 또는 MS-Word(.docx)
- **포함 내용**:
  - [ ] 게임 줄거리/시나리오 요약
  - [ ] 주요 캐릭터 설명 (체스 말, 바둑돌)
  - [ ] 게임 시스템 설명 (클릭, 강화, 동료, 보스)
  - [ ] 게임 조작 방법
  - [ ] 주요 게임 진행 장면 스크린샷 (5~10장)
  - [ ] 골드/루비 획득 방법
  - [ ] 게임 종료 방법

### 2. 플레이 동영상
- **분량**: 30분 내외
- **형식**: mp4, avi, mov 등
- **포함 장면**:
  - [ ] 연령등급 표시 화면
  - [ ] 게임 인트로/메인 화면
  - [ ] 바둑돌 클릭 및 파괴
  - [ ] 강화 시스템 (성공/실패)
  - [ ] 업그레이드 시스템
  - [ ] 동료(자동클릭) 시스템
  - [ ] 상점 시스템
  - [ ] 보스 전투
  - [ ] 체스 말 진화
  - [ ] 더보기 메뉴
  - [ ] 종료 확인 모달

### 3. 게임 빌드
- `npm run build` 실행하여 dist 폴더 생성
- 또는 배포된 URL 제공

## 등급분류 후 필수 표시 사항

게임 내에 다음 정보를 표시해야 합니다:

```
게임명: 바둑돌 부수기
등급: 전체이용가
등급분류번호: (심의 후 발급)
제작사: (입력)
```

### 표시 위치 및 방법
- **연령등급 마크**: 게임 시작 시 우측 상단에 3초 이상 표시
- **운영정보**: 더보기 메뉴 또는 설정에서 확인 가능하도록

---

## 주의사항

1. **개인도 신청 가능** - 사업자등록증 없이도 개인회원으로 신청 가능
2. **게임 완성 필수** - 심의 신청 전 게임이 완성된 상태여야 함
3. **내용 변경 시 재심의** - 게임 내용이 크게 변경되면 재심의 필요
4. **등급분류번호 필수 표시** - 발급 후 게임 내에 반드시 표시
5. **허위 신청 금지** - 설문 내용과 실제 게임 내용이 일치해야 함
